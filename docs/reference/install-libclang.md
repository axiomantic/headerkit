# Install Libclang

CLI tool for installing the libclang system dependency.

## CLI Usage

```
clangir-install-libclang [--version VERSION] [--skip-verify]
python -m clangir.install_libclang [--version VERSION] [--skip-verify]
```

See the [Installing libclang guide](../guides/install-libclang.md) for detailed usage, platform behavior, and CI examples.
